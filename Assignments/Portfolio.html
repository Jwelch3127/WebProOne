Final Project Description

My Final Assignment in the Web Programming I course, was to create a fully responsive webpage or simple website using HTML, CSS, and JavaScript. The other part was that the topic or content had to relate to education....For my final I decided to create a simple webpage that provides tips and in depth looks at five stress management strategies. The concept came about as I was stressed out by my finals (who isn't) as well as my own understanding of mental health (and how it affects me personally). Within the last few years, this country has suffered a great deal of hardship that has permiated every aspect of our lives. With expectations, news, media and this overwhelming world in which we exist, stress, I imagine, is on the rise - which makes it an epidemic (in my opinion). This makes understanding stress and coping with it of the utmost importance. I wanted to provide content that can be used to benefit others. I wanted to use this project as an opportunity to not just apply the skills learned in this course, but to develope a useful and practical learning device.
